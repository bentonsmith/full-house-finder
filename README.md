# full-house

## Project setup
```
npm install
```

## Must have Vue CLI installed
[Vue CLI install](https://cli.vuejs.org/guide/installation.html)

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
